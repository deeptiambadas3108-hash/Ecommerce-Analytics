🔹 Project Overview

This project analyzes an e-commerce dataset (Olist Store Data) to extract meaningful business insights. It includes interactive dashboards built using Excel, Power BI, and Tableau, along with SQL scripts for data extraction, transformation, and analysis.

The goal of this project is to understand customer behavior, sales performance, payment trends, and operational efficiency.

🔹 Tools & Technologies Used
SQL (MySQL)
Excel (Dashboard & Data Analysis)
Power BI (Interactive Dashboard)
Tableau (Data Visualization)
🔹 Dashboards Included
📌 Excel Dashboard
Total Orders, Revenue, Customers, Sellers
Customer distribution by state
Revenue vs Order Volume
Weekday vs Weekend sales
Top product categories
Payment type distribution
📌 Power BI Dashboard
Total Sales, Profit, Customers, Sellers
Monthly sales trends
Shipping days vs review score
Payment analysis (weekday/weekend)
City-wise analysis
Order review insights
📌 Tableau Dashboard
Sales distribution (weekday vs weekend)
Orders with review score & payment type
Average delivery time by category
Monthly sales analysis
Customer city insights with filters
🔹 SQL Analysis

The SQL script includes:

Data exploration queries
Joins across multiple tables (customers, orders, payments, products)
Aggregate functions (SUM, AVG, COUNT)
Group By & Having clauses
Subqueries
Window functions (RANK, DENSE_RANK, ROW_NUMBER)
Views creation
Stored procedures
Indexing & constraints
🔹 Key Insights
Majority of payments are made using credit cards
Weekday sales are significantly higher than weekend sales
Certain product categories generate the highest revenue
Delivery time impacts customer review scores
Top cities and states contribute most to sales

🔹 How to Use
Run the SQL script to explore and analyze the dataset
Open dashboards in respective tools:
Excel → .xlsx
Power BI → .pbix
Tableau → .twbx / Tableau Public
Interact with filters and visuals to explore insights

🔹 Conclusion
This project demonstrates end-to-end data analytics workflow:
Data Extraction → Data Analysis → Data Visualization → Business Insights
